# **App Name**: CodeTalk

## Core Features:

- User Authentication: Secure user login and registration functionality.
- Chat List: Display a list of active chat conversations with the most recent message and timestamp.
- Real-time Messaging: Send and receive text messages in real time.
- User Status: Indicate user online/offline status.
- Settings: Allow users to modify profile and change password.
- AI Code Assistant: Provides suggestions for user's current work by using an LLM, triggered through a dedicated tool.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) for a calm, modern feel.
- Background color: Off-white (#F5F5F5) to provide a clean backdrop.
- Accent color: Darker blue (#4682B4) for interactive elements to draw focus.
- Body and headline font: 'Inter' (sans-serif) for a clean, readable interface.
- Use simple, line-based icons for options in the settings and chat functionalities.
- Employ a minimalist layout with consistent spacing to maintain visual clarity and reduce clutter.
- Incorporate subtle animations for message delivery and user status changes to provide a polished feel.
